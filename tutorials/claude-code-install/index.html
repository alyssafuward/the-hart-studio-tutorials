 Install on Mac  ·  Step 1 of 4

Open Terminal

Press ⌘ + Space to open Spotlight, type Terminal, and press Enter. It appears at the top of results with a black icon.

If you see iTerm in the list, that also works — it is just an alternative Terminal app.
1 / 4
 Install on Mac  ·  Step 2 of 4

Run the install command

Copy and paste the command below into Terminal and press Enter. This downloads and installs Claude Code in one step.

curl -fsSL https://claude.ai/install.sh | bash
Think of this like clicking Download on a website, except through Terminal instead of a browser.
2 / 4
 Install on Mac  ·  Step 3 of 4

Confirm the install

You will see Claude Code successfully installed! in green. You may also see a yellow PATH warning — if so, go to the next step.

If you did not see the warning, skip the next step. You are already good.
3 / 4
 Install on Mac  ·  Step 4 of 4

Fix the PATH (if needed)

If you saw the PATH warning, paste this command into Terminal and press Enter. It runs silently — no output means it worked.

echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.zshrc && source ~/.zshrc
After running that, open a new Terminal window before continuing.

⊞ Install on Windows  ·  Step 5 of 7

Open PowerShell as Administrator

Windows PowerShell (Administrator)─ □ ✕

PS C:\Users\YourName> [Environment]::SetEnvironmentVariable('Path', $env:Path + ';C:\Users\YourName\.local\bin', 'User')

Open PowerShell as Administrator: press the Windows key, type PowerShell, then right-click and choose Run as administrator. Click Yes when Windows asks for permission.

This elevated window is only needed to fix the PATH. Once done you will close it and never need it again for Claude Code.

⊞ Install on Windows  ·  Step 6 of 7

Fix the PATH

Windows PowerShell (Administrator)─ □ ✕
Windows PowerShell
Copyright (C) Microsoft Corporation. All rights reserved.
 
PS C:\Users\YourName> [Environment]::SetEnvironmentVariable('Path', $env:Path + ';C:\Users\YourName\.local\bin', 'User')
 
PS C:\Users\YourName>

In the Administrator PowerShell, paste this command and press Enter. Replace YourName with your actual Windows username.

[Environment]::SetEnvironmentVariable('Path', $env:Path + ';C:\Users\YourName\.local\bin', 'User')
It runs silently — no output means it worked. Close this Administrator window when done, then open a regular PowerShell to continue.

⚙ Claude Code Setup  ·  Step 1 of 15

Launch Claude Code

Type claude in your terminal and press Enter. Claude Code will start and ask how you want to log in.

Choose option 1: Claude account with subscription to connect to your existing claude.ai account.

If you have a Pro, Max, Team, or Enterprise Claude subscription, option 1 is the right choice.
1 / 15
⚙ Claude Code Setup  ·  Step 2 of 15

Authorize in your browser

A browser window will open asking you to authorize Claude Code. This is the same kind of permission screen you see when logging in with Google. Click Authorize.

This is safe — you are just linking your existing Claude account to the terminal version.
2 / 15
⚙ Claude Code Setup  ·  Step 3 of 15

Allow network access

Your computer may show a security dialog asking if your terminal can access the network. Click Allow. Claude Code needs this to communicate with Anthropic.

On Windows this may look different but works the same way.

⚙ Claude Code Setup  ·  Step 4 of 15

Login confirmed in browser

You will see this page in your browser. It means the login worked. Close that tab and go back to your terminal.

⚙ Claude Code Setup  ·  Step 5 of 15

Login confirmed in terminal

Back in your terminal, you will see your email address with Login successful. Press Enter to continue.

⚙ Claude Code Setup  ·  Step 6 of 15

Read the security notes

Claude Code shows two reminders:

1. Claude can make mistakes. Always review what it does, especially when it touches files.

2. Only use it with code you trust. Do not point it at unknown files from the internet.

Press Enter to continue.

Worth reading once and keeping in mind.
6 / 15
⚙ Claude Code Setup  ·  Step 7 of 15

Choose your theme

Claude Code asks you to pick a visual theme. Use your arrow keys to scroll — the preview at the bottom updates as you go.

If your terminal has a dark background, choose Dark mode. If it is white or light gray, see the next slide.

You can change your theme any time by typing /theme inside Claude Code.

⚙ Claude Code Setup  ·  Step 8 of 15

Light mode option

Here is Light mode — easier to read on a white or light terminal background. Pick whichever looks clearest and press Enter.

Dark mode = Monokai Extended. Light mode = GitHub syntax.
8 / 15
⚙ Claude Code Setup  ·  Step 9 of 15

Terminal setup

Claude Code asks if you want recommended terminal settings. This enables Option+Enter for newlines and a visual bell. Choose Yes, use recommended settings.

You can undo later with /terminal-setup.
9 / 15
⚙ Claude Code Setup  ·  Step 10 of 15

Trust this folder

Claude Code asks if you trust the folder it is accessing. Since you are in your own home folder, choose Yes, I trust this folder and press Enter.

If you ever launch Claude Code in an unfamiliar folder, it is fine to say No and exit.

⚙ Claude Code Setup  ·  Step 11 of 15

Fullscreen renderer

Claude Code may ask if you want to try the new fullscreen renderer — smoother output, mouse support, and text auto-copies when selected. Choose Yes, try it.

After this you will see the main Claude Code dashboard.
11 / 15
⚙ Claude Code Setup  ·  Step 12 of 15

Permission popups

Your computer may show permission dialogs asking Claude Code to access different folders. Click Allow for anything you want it to work with. It is fine to click Don't Allow for anything you want to keep off-limits.

Photos has three options — unless you plan to work with your photo library, Don't Allow is fine.

⚙ Claude Code Setup  ·  Step 13 of 15

The main interface

You will land on the Claude Code home screen with your account info and a prompt at the bottom. The prompt is where you type. Try saying hi.

The left panel shows your account. The right panel shows tips and release notes.

Claude is thinking

After you send a message, you will see Thinking... while Claude processes. Press Esc at any time to stop.

Claude responds — you are ready!

Claude responds right in your terminal. You are now fully set up.

Type ? to see available shortcuts, or just start asking Claude what you need.
